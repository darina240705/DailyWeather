# DailyWeather

Ибраева Дарина Жалгасовна  
9-2-ПО-22  

Приложение DailyWeather приложение, предназначенное для отслеживания погоды. С его помощью можно просмотривать погоду на несколько дней вперед и планировать каждый выход заранее. Так люди смогут с меньшей вероятностью попасть под плохую погоду. У данного приложения, из-за специфики, нет конретной целевой аудитории, так как смотреть погоду могут люди разных возрастов, уровня жизни и рода деятельности.
